# Code Examples

Worked examples to read alongside the specification.  Each is an interactive notebook—click the rocket icon in the page header to launch the live kernel.

* [**Nouns and Data**](./nouns-and-data.ipynb) — how everyday data shapes (signed integers, dates, lists, vectors, tagged unions, optionals) get encoded as Nock nouns.
* [**Idioms, Gates, & Design Patterns**](./idioms-gates-patterns.ipynb) — recurring Nock idioms for the patterns Hoon calls gates, cores, and doors.
* [**Nock for Everyday Coders**](./timluc-miptev--nock-for-everyday-coders.ipynb) — ~timluc-miptev's full-length introductory tutorial, originally published in 2020, walking through Nock from first principles for working programmers.
